# Título do Projeto Extremamente Aesthetic ;)

Humano ou IA?

## 📒 Descrição
Utilizar uma IA para criar um texto.
Utilizar outra IA para criar um avatar que leia e fale o texto. 

## 🤖 Tecnologias Utilizadas
ChatGPT
